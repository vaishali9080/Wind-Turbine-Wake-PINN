# Physics-Informed Neural Network for Wind Turbine Wake Flow Prediction

## Overview
Developed a Physics-Informed Neural Network (PINN) using PyTorch to predict velocity deficit from experimental wind tunnel wake measurements.

## Features
- Data preprocessing using Pandas
- PINN implemented in PyTorch
- Training on 1D & 3D wake data
- Validation on 2D wake data
- Performance evaluation using RMSE, MAE and R²

## Tools
Python, PyTorch, Pandas, NumPy, Scikit-learn, Matplotlib, Google Colab
